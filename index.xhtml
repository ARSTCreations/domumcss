● Deployment

- domumCSS via cdn -
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/ARSTCreations/domumcss@main/domum.min.css" />
  <script src="https://cdn.jsdelivr.net/gh/ARSTCreations/domumcss@main/domum.min.js"></script>

- FontAwesome via cdnjs (or whatever cdn you're using) -
  <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/4.7.0/css/font-awesome.min.css" />

- and this script in the end if using on scroll animation
  <script> initanima(); </script>

- Once Deployed, your web goes Dark Mode -

● Listing

Lists are better formatted with the class class="list" and class="descList" to describe the lists.

<dl class="descList">
 <li><a href="#">Category 1</a></li>
 <dd><a href="#">Subcat 1.2</a></dd>
 <li><a href="#">Category 2</a></li>
 <dd><a href="#">Subcat 2.2</a></dd>
</dl>
<ul class="list">
 <li><a href="#">Category 3</a></li>
 <li><a href="#">Category 4</a></li>
 <li><a href="#">Category 5</a></li>
 <li><a href="#">Category 6</a></li>
 <li><a href="#">Category 7</a></li>
</ul>

The result will be like the following

  • Category 1
    • Subcat 1.2
  • Category 2
    • Subcat 2.2
    • Category 3
    • Category 4
    • Category 5
    • Category 6
    • Category 7
